# Galaxy frontend

This folder is a Vite + React frontend target. The repository already contains a working `src/` at the project root.

To run the frontend from `new/frontend`: copy the `src/` folder and `index.html` from the project root into this folder, then:

```bash
cd new/frontend
npm install
npm run dev
```

Or you can run the existing app from the repository root (it already has the full frontend).
